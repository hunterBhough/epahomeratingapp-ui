User Guide

The user guide is organized into three sections:

RaterPRO Desktop – Learn how to use RaterPRO’s web interface to create and review jobs.

RaterPRO Mobile App – Learn how to use RaterPRO’s mobile app to perform inspections in the field.

Providers – As a quality assurance provider, learn how to review jobs submitted by rating companies.

Before diving into the user guide, it’s important to understand the three types of users around which RaterPRO’s workflow is structured:

Rater Admins are office staff who create jobs and review inspection results. They also control user permissions and invitations for the organization’s account. Rater Admins use the RaterPRO Desktop site.

Raters are HERS raters who perform inspections using the RaterPRO mobile app, edit inspection checklist items, and make the final decision to mark a job as ‘complete.’ Raters spend most of their time on the mobile app.

Providers review submitted jobs from rating companies. They also control user permissions. Providers use the RaterPRO Desktop site.

RaterPRO jobs start and end in the energy modeling software. In between, RaterPRO’s core workflow is:

  1. Rater Admin creates a job in RaterPRO by importing the XML from energy modeling software.
  2. Rater performs inspections using the mobile app.
  3. Rater ‘completes’ the job once all items are satisfactory.
  4. Rater Admin reviews the job.
  5. Rater Admin exports the job to energy modeling software (via XML file) and calculate the final HERS score.
  6. Optionally, Rater Admin shares job with Provider who refers to job records during their QA review.

The guide will explain each of these steps in detail. If you have any further questions, please contact the ENERGY STAR team at energystarhomes@energystar.gov.
